# Auto-Caller
Auto-Caller project


## 🚀 Getting Started

### ⚛️ 🖥️ Frontend (React)

```bash
cd frontend
npm install
npm start
```

### ⚛️ 🖥️ Backend (Flask)
```bash
cd backend
python3 -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate
pip install -r requirements.txt
flask run
```
